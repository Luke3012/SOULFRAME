# PS2 Theme Spec (UI Layer)

## Render Pipeline
- **Detected**: URP (GraphicsSettings points to a Universal Render Pipeline asset). 
- **WebGL**: Keep post effects light. Use Bloom only for the glow layer.

## Palette
| Token | Hex | Use |
| --- | --- | --- |
| PS2 Black | `#05070B` | Background/Canvas |
| Deep Navy | `#0B1230` | Panels, overlays |
| PS2 Cyan | `#5FD6FF` | Highlights / selection |
| Soft Cyan | `#9BE7FF` | Secondary glow |
| Steel Gray | `#A9B0BF` | Body text |
| Slate Gray | `#6C768C` | Disabled text |

## Typography (TMP)
- **Primary**: TMP SDF (e.g. *Inter*, *Eurostile*, *Oxanium*). 
- **Outline**: subtle `0.08–0.12` with dark navy to simulate PS2 edge.
- **Glow**: Use TMP `Glow` (inner 0.05–0.08, outer 0.12–0.2, power ~0.6).
- **Spacing**: Character spacing `2–4`, line spacing `-5` for tight PS2 feel.

## Layout Rules
- **Aspect**: 16:9 target.
- **Canvas Scaler**: `Scale With Screen Size`, `Reference Resolution = 1920x1080`, `Match = 0.5`.
- **Safe Area**: keep the hint bar within 5% bottom margin on all panels.

## Background Rings (PS2 Dots)
- **GO**: `VFX_BackgroundRings`
- **Implementation**: `PS2BackgroundRings` creates a ParticleSystem at runtime.
- **Material**: `PS2_RingAdditive` (URP additive shader). If no texture is assigned, the script generates one at runtime.
- **Behavior**: low speed drift, long lifetime, soft fade-in/out.

## Selection Blink
- **Component**: `SelectableBlink` attached to TMP text or Image.
- **Behavior**: Cyan pulse using `Time.unscaledTime`, with TMP glow for emphasis.

## Hint Bar
- **GO**: `UI_HintBar`
- **Text**: TMP text with inline icons or Unicode fallback.
- **Data**: Use `UIFlowController` hint entries to swap per state.

# Asset Import / Pipeline

## Icons (PS2-like)
**Option A (CC0)**
- [Kenney Game Icons](https://kenney.nl/assets/game-icons) (CC0). Safe for commercial use.

**Option B (Free, attribution required)**
- [Iconic](https://iconic.app/) or [Font Awesome Free](https://fontawesome.com/) (CC BY 4.0 for some packs; check license). 

### SVG → Unity (Vector Graphics)
1. Install **Vector Graphics** package.
2. Import SVGs into `Assets/Art/Icons`.
3. In Inspector: set **Pixels Per Unit** (e.g. 100), **Scale** = 1.
4. Use them as Sprite in Image or TMP Sprite Asset.

### SVG/PNG → TMP Sprite Asset (inline icons)
1. Prepare a single atlas PNG (`1024x1024`) with icons (white on transparent).
2. In Unity: **Create → TextMeshPro → Sprite Asset**.
3. Assign atlas, set **Default Sprite Size** (e.g. 64).
4. Click **Generate Sprite Asset**.
5. Use in TMP text: `<sprite name="Cross"> Enter`.

Reference: https://docs.unity3d.com/Packages/com.unity.textmeshpro@3.2/manual/Sprites.html

# Ring Texture (optional download)
You can keep the procedural texture generated by `PS2BackgroundRings`, or drop in a custom ring PNG:
1. Download a free ring sprite (example pack: **Kenney Particle Pack** – CC0).  
2. Extract a circular ring sprite and save it as `Assets/Art/Textures/PS2_Ring.png` (white ring on transparent background).
3. Assign it to `VFX_BackgroundRings` → **Ring Texture** (or set it in the prefab).

# Unity Settings (Recommended)

## Post Processing (URP)
- Add **PS2PostProcessingBootstrap** to the canvas root (already wired).
- Suggested Bloom: `Intensity 0.8–1.0`, `Threshold 0.7–0.8`, `Scatter 0.6–0.8`, `Tint #B3E8FF`.

## TMP Materials
- Enable **Glow** and tweak per selection.
- Use **Outline** with dark navy for PS2 edge.

## Additive Ring Material
- Shader: `SOULFRAME/PS2AdditiveParticle`
- Color: `#99DFFF` w/ alpha 0.9

# Wiring Map (Inspector)

## UIFlowController
- `Hint Bar` → `UI_HintBar`
- `Hint Entries` → map per state:
  - MainMenu: `[X] Enter   [O] Back`
  - SetupVoice: `[X] Record   [O] Back`
  - SetupMemory: `[X] Save   [O] Back`
  - AvatarReady: `[X] Play   [O] Close`

## VFX_BackgroundRings
- `Ring Material` → `PS2_RingAdditive`
- `Ring Texture` → optional (assign custom PNG if you want to override the procedural ring)

## UI_HintBar
- `Hint Text` → TMP child component
